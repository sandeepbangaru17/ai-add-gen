# 🌐 AI Ad Creation Pipeline - Web Application

## Coming Soon!

This folder will contain the web application for the AI Ad Creation Pipeline.

### What We're Building

A web application that lets users:
1. Enter campaign details through a form
2. Generate professional video ads in real-time
3. Preview and download videos in multiple formats

### Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React + Vite + Tailwind CSS |
| Backend | FastAPI (Python) |
| Database | SQLite |
| Storage | Local disk / S3 |

### Quick Start (Tomorrow)

```bash
# 1. Start Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

# 2. Start Frontend (new terminal)
cd frontend
npm install
npm run dev
```
